# PythonLibrary
fatimaismail-utils is a Python utility library developed for an Introduction to Data Science course. It is designed to streamline common programming tasks by providing a modular set of tools for numerical calculations and string manipulation

## Key Modules & Features
The library is divided into two primary components to ensure clean, reusable code:
1.Calculator Module: Handles arithmetic operations including multiplication, division (with error handling for zero), power calculations, percentages, and calculating the mean of a list.
2.Formatter Module: Provides text styling and processing tools, such as generating personalized greetings, wrapping text in Markdown bold syntax, reversing strings, counting words, and title-casing text.

## Professional Integration
